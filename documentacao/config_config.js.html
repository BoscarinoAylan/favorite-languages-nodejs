Visão Geral

Esse módulo possui os dados de configuração que serão utilizados por certas partes da aplicação, por exemplo dados de conexão com o banco de dados. O módulo trata-se de exportar um objeto que tem dois atributos principais que exportam as configurações específicas para o ambiente de desenvolvimento e produção.

Os valores do atribtudo de desenvolvimento (development) podem variar de acordo com as configurações de cada computador podendo então ser alterados de acordo com as suas necessidades. Os valores do atributo de produção (production) são recuperados das variáveis de ambiente do servidor de produção e não devem ser alterados a não ser que o servidor utilize variáveis homônimas para outros fins. quando puser a aplicação em produção configure os valores nas variáveis de ambiente para que se adequem com as necessidades da aplicação.
