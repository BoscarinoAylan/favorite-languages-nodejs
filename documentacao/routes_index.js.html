src/routes/index.js

const Router = require('express').Router();
const apiRouter = require('./api');

Router.use('/api', apiRouter);

module.exports = Router;

Visão geral

Esse é o módulo padrão para gerenciar as rotas da aplicação, nesse caso a aplicação fornece somente uma API então ela irá direcionar para o módulo que irá gerenciar as rotas do endpoint /api.

Comentários

  • Linha 1: Armazenamos o objeto Router do express dentro de uma constante. Atravéz desse objeto nós iremos definir os eventos do ciclo requisição-resposta da aplicação. 
    const Router = require('express').Router();
  • Linha 2: Importamos a rota que será responsável por gerenciar os endpoints da API. 
    const apiRouter = require('./api');
  • Linha 4: Utilizamos o método Router.use() para associar todas as requisições, independente do verbo HTTP, direcionadas ao endpoint /api com seu devido módulo de rotas. 
    Router.use('/api', apiRouter);
  • Linha 6: Exportamos o objeto Router para que o módulo de rotas possa ser usado pela aplicação. 
    module.exports = Router;
