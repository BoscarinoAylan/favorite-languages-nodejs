Esta aplicação é uma api de linguagens de programação com autenticação de usuários, a API permite cadastrar um novo usuário, logar em um usuário já existente, deslogar um usuário que esteja logado, listar todas as linguagens de programação cadastradas, exibir os detalhes de uma linguagem específica e curtir uma linguagem de programação específica

Dependências da aplicação:

Como estamos utilzando o NPM, podemos encontrar todas as dependências da nossa aplicação no arquivo package.json. Essa informação é crucial pois, caso instalemos os pacotes nas versões erradas, podemos ter sérios problemas de incompatibilidade. Nesse projeto temos como dependências os seguintes pacotes: 

"cors": "^2.8.5",
"email-validator": "^2.0.4",
"express": "^4.16.4",
"jsonwebtoken": "^8.4.0",
"mongoose": "^5.3.14",
"password-validator": "^4.1.1"

Destaques da aplicação

  • API RESTful em Node.js
  • Express
  • JSON Web Token
  • Mongoose ORM
  • MongoDB
  • Criptografia de senha
