Esta aplicação é uma api de linguagens de programação com autenticação de usuários, a API permite cadastrar um novo usuário, logar em um usuário já existente, deslogar um usuário que esteja logado, listar todas as linguagens de programação cadastradas, exibir os detalhes de uma linguagem específica e curtir uma linguagem de programação específica

Dependências da aplicação:

  • cors
  • email-validator
  • express
  • jsonwebtoken
  • mongoose
  • password-validator

Destaques da aplicação

  • API RESTful em Node.js
  • Express
  • JSON Web Token
  • Mongoose ORM
  • MongoDB
  • Criptografia de senha
